# Employee Tracker 

## User Story

```
AS AN data manager for my company
I WANT to manage my coworkers according to their roles, deparment and salary
SO THAT I can give out appropriate data to the CEO of my company
```

## Acceptance Criteria

```
GIVEN I have completed a data table for my company
WHEN I run the application
THEN THE following options will appear: view department, view role, view employee
WHEN I view the department
THEN A data table will present the department that the employee work in
WHEN I view the role
THEN A data table will present the job title, salary and department id
WHEN I view the employee
THE A data table will present the employee first name, last name, role id and manager id
```
